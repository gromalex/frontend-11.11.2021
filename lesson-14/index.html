Что такое сеть?

1. Чтобы соединить 2 компьютера в сеть необходимы:
  • Ethernet кабель
  • WiFi
  • Bluetooth

Рассмотрим простейший способ с использованием Ethernet кабеля

2. С каждым подключаемым компьютером сеть усложняется:

Для решения этой проблемы каждый компьютер сети подключается к одному компьютеру, который называется маршрутизатор.

Сеть упрощается:

3. Сеть сетей

Для объединения сотни,тысячи или миллионов компьютеров один маршрутизатор не справится с этой задачей.

Так как маршрутизатор - это специальный компьютер, то их тоже можно соединить.

4. Модем

Такая сеть уже похожа на то, что мы называем интернетом, но мы не можем протянуть кабели по всему миру.

Мы можем воспользоваться уже готовой сетью, которая соединяет разные уголки мира. Эта сеть - телефонная.

Для подключения компьютерной сети к глобальной телефонной необходимо оборудование, которое перекодирует информацию, поступающую из нашей сети в формат, который можно передавать через телефонную сеть, и, наоборот, декодируют информацию из телефонной сети в формат, который распознают наши компьютеры. Для этого нужен модем.

5. ISP (Internet Service Provider)

ISP - компания, которая обслуживает специальные маршрутизаторы, которые не только подключены друг к другу (объединяют в единую сеть всех клиентов провайдера), но также связаны с маршрутизаторами других провайдеров. Таким образом, наше сообщение, пройдя транзитом через сеть нескольких провайдеров, достигнет сеть назначения.

Интернет — это сеть сетей, которая объединяет в себе всю вышеперечисленную инфраструктуру.

6. Поиск компьютера в сети происходит по IP (Internet Protocol)

Пример интернет протокола версии 4 (IPv4) 192.168.2.10 или IPv6 2027:0da8:8b73:0000:0000:8a2e:0370:1337

Человеку сложно использовать IP, для него существуют домены, например: google.com

0.0.0.0 - 255.255.255.255
